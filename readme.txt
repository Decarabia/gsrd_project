*GSR Data Project*

*Program Files Description*

grs_stats_II.py: main menu

append.py: Combines all data in gsr_excel_files into one file called gsrd.xls

data_selection.py: Submenu for choosing data sets.

freq_II.py: Submenu for working with gsrd.xls

quarterly_stats_II.py: Computes stats by quarter.

year_stats.py: Computes stats by year.

term_stats.py: Computes stats be season.

stats_proc.py: Contains all functions for calculating confidence intervals and means.

*Other Files*

anova.py: ANOVA using different data sets.

corr_matrix.py: Correlation matrix.

regression.py: Multiple regression script.
